Featured Projects

Exium

   

I built Exium, a secure coding exam platform with ReactJS and TailwindCSS featuring lockdown mode, activity monitoring, and Monaco Editor. Includes offline sync and automated grading through Judge0 API on AWS EC2.

Check it out!

Manimator

   

I developed Manimator, a web application that enables users to create mathematical animations programmatically using Manim. Built with Flask and ReactJS, it features an intuitive interface that leverages the OpenAI API to generate Manim code based on user input and renders animations in real-time. The platform streamlines the animation creation process while maintaining Manim's powerful capabilities.

Check it out!

CheckED

CheckED is a cross-platform mobile event planning application developed using .NET MAUI (Multi-platform App UI). The app streamlines event organization by providing an intuitive interface for both event organizers and general users. Event organizers can create and manage events, while general users can easily discover and access event information through the application's user-friendly interface.

Check it out!

Sudoku

A modern Sudoku puzzle solver application featuring a sleek dark-themed graphical user interface built using Qt6 and C++17. The solver implements an efficient Ternary Search Tree algorithm to quickly solve Sudoku puzzles of varying difficulty levels. The application provides an intuitive interface for users to input puzzles and visualize the solving process in real-time.
